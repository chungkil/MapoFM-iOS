# 마포FM iOS 버전 제안
현재 '씨알'이라는 이름으로 안드로이드 버전만 존재하는 마포FM의 앱을 iOS용으로 확장하기 위한 제안용 앱 프로젝트

## 목표
- [마포FM씨알 안드로이드 버전](https://play.google.com/store/apps/details?id=net.mapofm.musicapp)과 동일한 기능 제공
- iOS에 적합한 UI로 재구성
- 실효성 있는 기능으로 재구성
